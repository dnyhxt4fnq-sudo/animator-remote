# Animator Remote

**Edit, drive and monitor Unity Animator Controllers live from your phone or tablet — over your local Wi‑Fi, in any mobile browser.**

A local web server runs inside the Unity Editor and serves a touch UI to any device on the same network. No app install, no cloud.

- **Edit Mode** — full graph editor: states, transitions, conditions, parameters, assign motions. Fully Undo/Redo‑able.
- **Play Mode** — drive parameters & triggers on the running character and watch the active state + progress update live.

**Platforms:** Unity 6 (6000.x)+ · Editor on Windows / macOS / Linux · Device: iPhone / iPad / Android

Available on the Unity Asset Store.
